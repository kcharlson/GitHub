INTRODUCTION

Cascading Style Sheets (CSS) is a stylesheet language used for describing the look and style of a document written in a markup language. Along with HTML and JavaScript, CSS is a cornerstone technology used by most websites to create visually engaging web pages, and apps.

CSS is designed primarily to enable the separation of document content from document presentation, including aspects such as the layout, colors, and fonts. The separation makes it easier to code, provide more flexibility and control in the specification of characteristics, enable multiple HTML pages to share formatting. Think of it like paragraph/character styles InDesign. It helps establish a consistent look to all pages, and it makes it easier to edit the appearance of a website quickly.
