The money symbol ($) is the key for all this to work. For the browser to understand what you are trying to do, it needs the $, plus a CSS selector (class or ID) in parentheses and quotes (single or double will both work). This combation turns the CSS element into a jQuery element, and allows it to be combined with functions and actions.

Basic event structure

$('selector, class or id').event();

without the "$" symbol, the browser with not understand that you are trying to create/use a jQuery action. Note you must put your selector within a pair of quotes.
$('button').fadeIn('fast');

In this example, the selector is a button element. The fadeIn event has been attached, witch means when the pages loads, the element will fade-in the hidden element.
$('.code').css("background-color","#232323");

In this example, the selector is a element with the class of '.code'. The CSS event has been attached, this will change the element's background color to #232323 by creating a inline style within the element's html code.
